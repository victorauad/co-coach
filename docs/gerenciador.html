O gerenciador usa a GitHub API para ler e salvar arquivos diretamente no repositório. Configure uma vez — fica salvo no browser.

Crie em GitHub → Settings → Developer settings → Personal access tokens → Fine-grained. Permissões necessárias: Contents: Read & Write nos repos co-coach, iracing_analysis, alamtoco e SMPL.

Sync de Repos

Quais repositórios recebem as skills automaticamente.

config/sync-targets.yml

Formato YAML: uma lista de owner/repo por linha. O sync-skills.yml distribui para todos os repos listados.

Como funciona o sync

1

Você cria ou edita uma skill em skills/co-coach-*/SKILL.md

2

O git push dispara automaticamente o workflow sync-skills.yml

3

O workflow lê config/sync-targets.yml e envia a skill para cada repo listado via GitHub API

✓

A skill fica disponível em .claude/skills/ de cada repo — sem precisar copiar manualmente
