# Remote MCP Servers

Servidores MCP remotos permitem que desenvolvedores se conectem via Anthropic MCP connector API a serviços externos sem precisar rodar servidores localmente.

## Como Conectar

1. Revisar a documentação do servidor específico
2. Obter as credenciais de autenticação necessárias
3. Seguir as instruções de conexão específicas de cada empresa/serviço

## Comportamento

Uma vez conectado, ferramentas MCP remotas seguem o mesmo comportamento de acionamento que qualquer outra ferramenta. O Claude decide quando usar com base na descrição da ferramenta e no contexto da conversa.

## Onde Encontrar Mais Servidores

- [GitHub — centenas de MCP servers](https://github.com/modelcontextprotocol/servers)
- [MCP connector docs](https://platform.claude.com/docs/en/agents-and-tools/mcp-connector)

## Aviso de Segurança

Servidores MCP de terceiros não são de propriedade, operação ou endosso da Anthropic. Só conecte a servidores que você confie e revise as práticas de segurança e termos de cada um.
