# Introduction to Subagents — Curso Anthropic

Curso gratuito sobre como usar e criar sub-agentes no Claude Code para gerenciar contexto, delegar tarefas e construir workflows especializados.

## O Que Você Vai Aprender

- **Funcionamento de sub-agentes**: como o Claude Code cria uma janela de contexto separada, fluxo de entradas e retorno de resumos
- **Criação de sub-agentes personalizados**: uso do comando `/agents` para construir assistentes adaptados ao seu workflow
- **Design eficaz de sub-agentes**: padrões para confiabilidade, formatos estruturados e limitação de ferramentas
- **Quando utilizá-los**: orientação prática sobre casos de uso apropriados e anti-padrões a evitar
