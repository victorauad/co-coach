# Apache Superset — Plataforma open-source de visualização de dados

**Tema:** ferramentas
**Fonte:** https://github.com/apache/superset
**GitHub Stars:** 73793

## Resumo

- Plataforma open-source de BI e exploração de dados da Apache Foundation — alternativa gratuita a Looker/Tableau/Power BI
- Conecta nativamente ao BigQuery (e a dezenas de outros bancos SQL) — dashboards direto sobre as tabelas, sem exportar dados
- Editor SQL embutido (SQL Lab) para explorar dados antes de montar o dashboard
- Dezenas de tipos de gráfico prontos, filtros interativos e dashboards compartilháveis por link
- Self-hosted: os dados não saem da sua infraestrutura — relevante quando há dados sensíveis de cliente/CRM

## Por que isso importa

Para análises recorrentes de Growth que hoje viram planilhas, o Superset transforma queries do BigQuery em dashboards vivos que o time consulta sozinho — reduz o retrabalho de gerar o mesmo relatório toda semana. Combina com a regra de escanear o mínimo de dados: a query otimizada vira a fonte do dashboard.
