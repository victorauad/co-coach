# Introduction to Agent Skills — Curso Anthropic

Curso gratuito da Anthropic no Skilljar sobre como construir, configurar e compartilhar Skills no Claude Code.

## O Que Você Vai Aprender

- O conceito de Skills e como diferem de outras opções de customização
- Criar sua primeira Skill do zero com frontmatter SKILL.md
- Desenvolver descrições eficazes que ativem correspondências confiáveis
- Organizar diretórios de skills com progressive disclosure
- Configurações avançadas como restrição de ferramentas (`allowed_tools`)
- Compartilhar skills com equipes via repositório
- Distribuir via plugins e configurações gerenciadas
- Integrar skills em subagentes personalizados
- Diagnosticar e resolver problemas comuns

## Módulos

1. O que são skills?
2. Criando sua primeira skill
3. Configuração e skills multi-arquivo
4. Skills vs. outras funcionalidades do Claude Code
5. Compartilhando skills
6. Resolução de problemas com skills
