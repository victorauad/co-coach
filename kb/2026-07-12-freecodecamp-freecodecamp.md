# freeCodeCamp — Currículo open-source de programação e computação

**Tema:** ferramentas
**Fonte:** https://github.com/freeCodeCamp/freeCodeCamp
**GitHub Stars:** 451655

## Resumo

- Plataforma gratuita e open-source com currículo completo de programação, matemática e ciência da computação — o repositório mais estrelado do GitHub
- Certificações práticas em trilhas relevantes para dados: JavaScript, Python, análise de dados com Python, machine learning e bancos de dados relacionais
- Aprendizado baseado em projetos: cada certificação exige construir projetos reais, não só assistir aulas
- Comunidade enorme (fórum, YouTube, publicação) com material em português
- Mantido por organização sem fins lucrativos — sem paywall em nenhum conteúdo

## Por que isso importa

Para quem usa Claude Code sem ser desenvolvedor, os fundamentos (lógica, Python, SQL) aumentam muito a qualidade do que se pede e se valida com IA. A trilha de análise de dados com Python é a mais alinhada ao trabalho com BigQuery e planilhas — útil como referência de estudo dirigido, não como pré-requisito.
