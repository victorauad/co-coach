# Introduction to Model Context Protocol — Curso Anthropic

Curso prático sobre construção de servidores e clientes MCP usando Python SDK, focando nos três primitivos principais: ferramentas, recursos e prompts.

## O Que Você Vai Aprender

- Arquitetura do MCP e como ele transfere responsabilidade de definição e execução de ferramentas para servidores especializados
- Sistema de comunicação agnóstico a transporte
- Fluxo completo requisição-resposta desde consultas do usuário até serviços externos
- Construir servidores MCP usando Python SDK com decoradores
- Implementar funcionalidade de gerenciamento de documentos
- Usar o MCP Server Inspector para testes e depuração
- Definir recursos para exposição de dados somente leitura
- Construir prompts com instruções pré-elaboradas
- Padrões de integração prática incluindo autopreenchimento e injeção de contexto

## Pré-Requisitos

- Conhecimento prático de Python
- Compreensão básica de JSON e padrões requisição-resposta HTTP

## Módulos

1. Introdução
2. Apresentação do MCP
3. Clientes MCP
4. Prática com servidores MCP
5. Implementação de cliente
6. Definição de recursos
7. Acesso a recursos
8. Definição de prompts
9. Prompts no cliente
10. Avaliação final
