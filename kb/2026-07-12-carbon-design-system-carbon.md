# Carbon — Design system open-source da IBM

**Tema:** ferramentas
**Fonte:** https://github.com/carbon-design-system/carbon
**GitHub Stars:** 9273

## Resumo

- Design system completo da IBM: componentes prontos, tokens de design, tipografia (IBM Plex) e diretrizes de acessibilidade
- Implementações oficiais para React, Web Components, Angular, Vue e Svelte — estética corporativa sóbria, ideal para produtos de dados
- Documentação exemplar de padrões de UI para dashboards, formulários e tabelas de dados densas
- Acessibilidade (WCAG) embutida nos componentes por padrão, não como esforço extra
- Alternativa ao Material UI quando o visual "Google" não combina com a marca

## Por que isso importa

Referência dupla: (1) para POCs e dashboards com cara profissional sem designer — os padrões de data table e visualização do Carbon são dos melhores documentados; (2) como modelo de como documentar um design system, útil quando um agente de IA precisa gerar UI consistente a partir de regras escritas.
