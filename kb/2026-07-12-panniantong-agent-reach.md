# Agent Reach — Dá ao agente acesso à internet inteira sem taxas de API

**Tema:** ferramentas
**Fonte:** https://github.com/Panniantong/Agent-Reach
**GitHub Stars:** 55292

## Resumo

- CLI que dá "olhos" ao agente de IA: lê e busca Twitter/X, Reddit, YouTube, GitHub, Bilibili e XiaoHongShu sem pagar APIs
- Resolve os bloqueios clássicos: transcrição de vídeos do YouTube, busca no Twitter sem API paga, Reddit sem 403, conteúdo atrás de login
- Escolhe, instala e testa automaticamente o melhor método de acesso para cada plataforma — quando um método quebra, a ferramenta troca por você
- Um único CLI em Python 3.10+, licença MIT, zero taxas de API
- Foi o repositório #1 em trending no GitHub (Trendshift)

## Por que isso importa

Para pesquisa de mercado e social listening em Growth, o custo de APIs (Twitter, por exemplo) costuma inviabilizar automações. Agent Reach zera esse custo e permite que agentes monitorem menções, analisem concorrentes e resumam vídeos — casos de uso diretos para as skills de social e competitor do co-coach.

## Citação

> Give your AI agent eyes to see the entire internet. Read & search Twitter, Reddit, YouTube, GitHub — one CLI, zero API fees.
