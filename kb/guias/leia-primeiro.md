# Leia primeiro

## O que é este repositório

Um lugar único pra você aprender e consultar como usar o Claude Code no trabalho. Não é um curso linear. É uma **base de conhecimento** — você abre o arquivo do que precisa no momento.

## A regra de ouro pra não travar

Você me disse que disciplina não é seu forte. Então a regra aqui é uma só:

> **Nunca abra este repositório só pra "estudar". Abra sempre com uma tarefa real em mente.**

Exemplo: "preciso cruzar dados de pipeline do HubSpot" → vai direto pro fluxo de MCP. Você aprende fazendo, e a teoria gruda porque tem onde encaixar. Estudar AI no abstrato, sem aplicar, é o caminho mais rápido pra desistir.

## Como abrir os arquivos no VS Code

- `Cmd + P` → digite o nome do arquivo → Enter.
- `Cmd + Shift + V` num arquivo `.md` → abre o **preview** formatado (bem mais agradável de ler).
- A barra lateral esquerda (`Cmd + Shift + E`) mostra a árvore de pastas.

## Sobre "aprender errado"

Você pediu pra eu cobrir os pontos cegos do seu raciocínio. Tem um arquivo dedicado a isso: `03-metodologias/pontos-cegos-do-meu-raciocinio.md`. Vale ler cedo — ele lista os erros mais comuns de quem está no seu estágio (consegue fazer as coisas seguindo tutorial, mas não entende o que aconteceu).
