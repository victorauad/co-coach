# Prompt Engineering — Visão Geral

Guia oficial da Anthropic para engenharia de prompts com Claude.

## Antes de Começar

Antes de otimizar prompts, você precisa de:
1. Critérios de sucesso claros para o seu caso de uso
2. Formas de testar empiricamente contra esses critérios
3. Um rascunho inicial de prompt para melhorar

Sem isso, prompt engineering é prematuro.

## Quando Fazer Prompt Engineering

Nem todos os problemas se resolvem com prompt engineering. Por exemplo:
- **Latência e custo** → às vezes melhor trocar o modelo
- **Qualidade de saída** → prompt engineering é o caminho certo

## Onde Aprender as Técnicas

Todas as técnicas (clareza, exemplos, XML, role prompting, thinking, prompt chaining) estão documentadas em:
- [Prompting best practices](https://platform.claude.com/docs/en/docs/build-with-claude/prompt-engineering/claude-prompting-best-practices)

O [Claude Console](https://platform.claude.com/) oferece ferramentas de prompt: gerador automático, templates com variáveis e melhorador de prompts.
