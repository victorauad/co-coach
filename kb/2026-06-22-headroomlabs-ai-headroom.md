# Headroom: Camada de Compressão de Contexto para Agentes IA

**Tema:** ferramentas
**Fonte:** https://github.com/headroomlabs-ai/headroom
**GitHub Stars:** 46410

## Resumo

- Reduz 60-95% de tokens consumidos em prompts, logs, RAG chunks e histórico de conversas antes de chegar ao LLM
- Funciona como library Python/TypeScript, proxy transparente, wrapper de agentes ou servidor MCP - sem mudanças de código
- Usa 6 algoritmos especializados (SmartCrusher, CodeCompressor, Kompress-base) com compressão reversível (CCR) e cache KV otimizado

## Por que isso importa

Para Heads de Growth em Martech, reduzir tokens em 60-95% impacta direto no custo por API call e na viabilidade financeira de automações com Claude Code. A compatibilidade com múltiplos agentes e o modo proxy zero-code facilitam implementação rápida sem reescrever fluxos existentes.

## Citação

> Headroom compresses everything your AI agent reads — tool outputs, logs, RAG chunks, files, and conversation history — before it reaches the LLM. Same answers, fraction of the tokens.
