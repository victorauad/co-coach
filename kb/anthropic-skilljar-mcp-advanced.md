# MCP Advanced Topics — Curso Anthropic

Curso avançado sobre recursos sofisticados e padrões de implementação para desenvolvimento de MCP em produção.

## O Que Você Vai Aprender

- **Amostragem (Sampling)**: como servidores MCP solicitam chamadas de LLMs através de clientes conectados
- **Notificações de Progresso e Logging**: sistemas de feedback em tempo real para operações longas
- **Acesso a Arquivos via Roots**: sistemas de permissão com limites de segurança por diretório
- **Arquitetura de Mensagens JSON**: especificação completa de mensagens MCP e comunicação bidirecional
- **Transporte Stdio**: comunicação via fluxos de entrada/saída padrão
- **Transporte StreamableHTTP**: uso de Server-Sent Events para HTTP
- **Limitações do HTTP**: impacto de flags de configuração
- **Escala em Produção**: quando usar HTTP stateless para escalabilidade horizontal
- **Seleção de Transporte**: como escolher conforme requisitos de deployment

## Pré-Requisitos

- Python async programming
- JSON e HTTP
- Server-Sent Events (SSE) básico
- Ter feito o curso Introduction to MCP (recomendado)
