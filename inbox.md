# Inbox de conteúdo

Cole aqui qualquer link que você queira absorver no repositório.
Um link por linha. Quando terminar, rode `/process-inbox` no Claude Code.

## Como usar

1. Cole o link abaixo (um por linha)
2. Opcional: adicione uma nota depois do link com `—` (ex: `https://... — foco na parte de agentes`)
3. No Claude Code, rode `/process-inbox`
4. O Claude vai buscar o conteúdo, extrair os aprendizados e atualizar os `.md` certos
5. Os links processados vão para a seção **Processados** automaticamente
